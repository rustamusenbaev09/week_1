# week_1

Мини-отчет по проекту Alfred
Цель проекта: разработка AI-агента для автоматической обработки электронной почты, определения спама и формирования черновиков ответов.
•	Архитектура: Workflow/State Machine на базе LangGraph с условными переходами.
•	Фреймворки: LangGraph, LangChain, ChatGroq, Langfuse.
•	Модель ИИ: Llama 3.1 8B Instant (Groq API).
•	Инструменты: IMAP (imaplib) для чтения Gmail, email для обработки писем, DuckDuckGo Search (smolagents), Langfuse для трассировки.
•	Функции: чтение писем, классификация (спам/обычное), генерация ответа, уведомление пользователя.
•	Достоинства: автоматизация обработки почты, модульная архитектура, интеграция с Gmail.
•	Интегрировал с личным Gmail почтой /n

Мини-отчет по проекту Alfred_2
Цель проекта: создание AI-ассистента Alfred для общения с пользователем, анализа изображений и использования внешних инструментов.

•	Архитектура: LLM Agent (ReAct) на базе LangGraph с графом состояний.
•	Основные фреймворки: LangGraph, LangChain, ChatGroq.
•	Используемая модель: Llama 3.1 8B Instant (через Groq API).
•	Инструменты: Python Tools (divide, extract_text), Base64, Google Colab userdata.
•	Функциональность: диалог с пользователем, вызов функций, распознавание текста на изображениях, обработка запросов.
•	Достоинства: модульность, расширяемость, поддержка Tool Calling и мультимодальности.
