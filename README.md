# 📅 Неделя 1

## 📌 Проект Alfred

### 🎯 Цель проекта
Разработка AI-агента для автоматической обработки электронной почты:
- определение спама;
- классификация писем;
- генерация черновиков ответов.

### 🏗 Архитектура

**Workflow (State Machine)** на базе **LangGraph** с условными переходами.

### 🛠 Используемые технологии

| Категория | Технологии |
|-----------|------------|
| Frameworks | LangGraph, LangChain |
| LLM | Llama 3.1 8B Instant |
| API | ChatGroq API |
| Monitoring | Langfuse |
| Дополнительно | IMAP (Gmail), DuckDuckGo Search |

### ⚙️ Основные возможности

- 📧 Чтение электронной почты
- 🚫 Определение спама
- 📂 Классификация писем
- ✍️ Генерация черновиков ответов
- 📬 Интеграция с Gmail

## 📌 Проект Alfred_2

### 🎯 Цель проекта

Создание AI-ассистента Alfred для общения с пользователем, анализа изображений и использования внешних инструментов.

### 🏗 Архитектура

**LLM Agent (ReAct)** на базе **LangGraph**.

### 🛠 Используемые технологии

| Категория | Технологии |
|-----------|------------|
| Frameworks | LangGraph, LangChain |
| LLM | Llama 3.1 8B Instant |
| API | ChatGroq API |
| Инструменты | divide(), extract_text() |
| Дополнительно | Base64, Google Colab |

### ⚙️ Основные возможности

- 💬 Общение с пользователем
- 🧠 Вызов Python-инструментов (Tools)
- 🖼 Анализ изображений
- 📝 Распознавание текста (OCR)
- 🤖 Обработка пользовательских запросов
